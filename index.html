LOG EVENTI
📖 Manuale

🗺 Mappa / Map

IT: Scorri con la rotella per zoomare. Trascina per spostarti. Doppio click per resettare la vista. Clicca su una rotta colorata per selezionarla.

EN: Scroll to zoom. Drag to pan. Double-click to reset. Click a coloured route to select it.

🔒 Rotte / Routes

IT: Le rotte si sbloccano completando missioni sui percorsi prerequisiti (e a volte pagando Reales). Il colore indica il livello di pericolo: 🟢 Verde → 🟡 Giallo → 🟠 Arancione → 🔴 Rosso. Vincere battaglie abbassa il pericolo.

EN: Routes unlock by completing missions on prerequisite routes (sometimes paying Reales too). Colour = danger level: 🟢 Safe → 🟡 Caution → 🟠 Dangerous → 🔴 Warzone. Winning battles reduces danger.

⛵ Missioni / Missions

IT: Seleziona una rotta → scegli una missione → assegna le navi → lancia. Le navi veloci riducono il tempo di viaggio. Alcune missioni richiedono cargo (Tabacco, Vino, Cacao). Le missioni completate avanzano nella barra progressi e sbloccano nuove rotte.

EN: Select route → pick mission → assign ships → launch. Faster ships shorten travel time. Some missions require cargo (Tobacco, Wine, Cocoa). Completed missions advance the progress bar and unlock new routes.

⚔ Combattimento / Combat

IT: Sulle rotte pericolose si combatte prima di partire. La probabilità di vittoria dipende dalla potenza della flotta (💥 Potenza + 🛡 Scafo) vs la forza nemica. Usa i 🔥 Fire Barrel per +12 potenza ciascuno. Sconfitta = nave danneggiata, missione annullata.

EN: Dangerous routes trigger combat before departure. Victory chance = fleet power (💥 Firepower + 🛡 Hull) vs enemy strength. Use 🔥 Fire Barrels for +12 power each. Defeat = ship damaged, mission cancelled.

⚓ Cattura navi / Ship capture

IT: Alcune rotte hanno una probabilità di catturare una nave nemica al completamento della missione. La nave catturata entra nella tua flotta (se c'è spazio nel porto).

EN: Some routes have a chance to capture an enemy ship on mission completion. The captured ship joins your fleet (if dock space is available).

🏪 Mercato / Market

IT: Acquista/vendi navi con Reales. Tipi disponibili: Gunboat, Schooner, Brig, Fregata, Man O'War — velocità vs potenza di fuoco vs capacità di carico. Acquista Fire Barrel con Gemmes (5 ◆ cad.).

EN: Buy/sell ships with Reales. Types: Gunboat, Schooner, Brig, Frigate, Man O'War — speed vs firepower vs cargo. Buy Fire Barrels with Gems (5 ◆ each).

⚙ Upgrade

IT: Clicca ⚙ su una nave ormeggiata per potenziare fino a 3 livelli ciascuno: 📦 Cargo, 💥 Potenza, 🛡 Scafo, 💨 Velocità. Il costo aumenta ad ogni livello.

EN: Click ⚙ on a docked ship to upgrade up to 3 levels each: 📦 Cargo, 💥 Firepower, 🛡 Hull, 💨 Speed. Cost increases per level.

💰 Risorse / Resources

  • ⚜ Reales — moneta principale / main currency
  • ◆ Gemmes — riparazione navi e Fire Barrel / ship repair & Fire Barrels
  • 🔥 Fire Barrel — bonus potenza in combattimento / combat power bonus
  • Cargo — Tabacco/Tobacco · Vino/Wine · Cacao/Cocoa — richiesto da alcune missioni / required by some missions

💾 Salvataggio / Save

IT: Il gioco si salva automaticamente dopo ogni azione. Usa 💾 Backup per esportare il salvataggio in JSON e 📂 Import per caricarlo. ⚠ Reset cancella tutti i progressi.

EN: Game saves automatically after every action. Use 💾 Backup to export your save as JSON and 📂 Import to load it. ⚠ Reset wipes all progress.
